## üìê Spesifikasi Teknis Detail

### Resolusi & Format
- **Resolusi**: 3840x2160 (4K UHD) - **Recommended**
- **Minimal**: 1920x1080 (Full HD)
- **Format**: PNG dengan transparansi (preferred) atau WebP
- **Aspect Ratio**: 16:9 (landscape) atau 1:1 (square) untuk versatility
- **Ukuran File**: Optimalkan untuk web (< 500KB jika mungkin, max 1MB)

### Desain Kartu
- **Ukuran**: 3.5 x 2 inches (proporsional dalam gambar)
- **Sudut**: Rounded corners, radius ~0.1 inches
- **Shadow**: Subtle drop shadow untuk elevation
- **Top Section**: 2/3 dari tinggi kartu
- **Bottom Section**: 1/3 dari tinggi kartu
- **Diagonal Divide**: Sharp angular zigzag/chevron pointing downward

### Warna (Hex Codes)
- **Gradient Top**: #667eea (modern purple) ‚Üí #764ba2 (purple-pink) ‚Üí #f093fb (pink)
- **Dark Navy Bottom**: #1a1a2e (modern dark navy instead of pure black)
- **Text Name**: #333333 (dark gray, bold)
- **Text Other**: #FFFFFF (white)
- **Logo**: #FFFFFF (white, high contrast)

### Tipografi
- **Font**: Arial atau Helvetica (clean sans-serif)
- **Nama**: Bold, 14pt, dark gray (#333333)
- **Valid**: Regular, 12pt, white (#FFFFFF)
- **Email/Telp**: Regular, 10pt, white (#FFFFFF)
- **Anti-aliasing**: Enabled untuk smoothness

### Spacing & Layout
- **Nama to Valid**: 0.2 inches vertical spacing
- **Email padding**: 0.15 inches from edges
- **Telp to Email**: 0.1 inches spacing
- **Logo size**: 20% of card height
- **Text alignment**: Nama & Valid centered, Email & Telp left-aligned

### Hand & Composition
- **Hand position**: Left or bottom-left of frame
- **Skin tone**: Neutral (light to medium, unisex)
- **Grip**: Thumb and index finger pinching, other fingers curled
- **Focus**: Hand sharply focused, not blocking text
- **Card tilt**: 20-30 degrees from horizontal

### Realism Elements
- **Paper texture**: Visible
- **Edge creases**: Faint
- **Gloss reflection**: Slight on surface
- **Fingerprint smudges**: Minor on edges
- **Depth-of-field**: Blur on background stack
- **Lighting**: Soft natural from above
- **Shadows**: Subtle for depth

## üîß Tips untuk Hasil Terbaik

### DALL-E (ChatGPT/DALL-E 3):
- Gunakan prompt versi singkat
- Request "high detail, professional photography style"
- Generate beberapa variasi dan pilih yang terbaik

### Midjourney:
- Gunakan parameter `--ar 16:9` untuk aspect ratio
- Tambahkan `--v 6` untuk versi terbaru
- Gunakan `--style raw` untuk hasil lebih realistis

### Stable Diffusion:
- Gunakan model realistis seperti "Realistic Vision" atau "DreamShaper"
- Set CFG scale: 7-9
- Set steps: 20-30 untuk detail lebih baik
