## üìê Spesifikasi Teknis

- **Resolusi**: 3840x2160 (4K) atau minimal 1920x1080 (Full HD)
- **Format**: PNG dengan transparansi (untuk background putih/transparan)
- **Aspect Ratio**: 16:9 atau 4:3 (sesuai kebutuhan)
- **Ukuran File**: Optimalkan untuk web (< 500KB jika mungkin)

## üîß Tips untuk Hasil Terbaik

### DALL-E (ChatGPT/DALL-E 3):
- Gunakan prompt versi singkat
- Request "high detail, professional photography style"
- Generate beberapa variasi dan pilih yang terbaik

### Midjourney:
- Gunakan parameter `--ar 16:9` untuk aspect ratio
- Tambahkan `--v 6` untuk versi terbaru
- Gunakan `--style raw` untuk hasil lebih realistis

### Stable Diffusion:
- Gunakan model realistis seperti "Realistic Vision" atau "DreamShaper"
- Set CFG scale: 7-9
- Set steps: 20-30 untuk detail lebih baik

## üì• Setelah Mendapatkan Gambar

1. **Optimasi Gambar:**
   - Kompres menggunakan [TinyPNG](https://tinypng.com) atau [Squoosh](https://squoosh.app)
   - Target: < 500KB untuk loading cepat

2. **Simpan dengan Nama:**
   - `business-card-stack.png` atau `business-card-stack.webp`

3. **Integrasi ke Website:**
   - Lihat file `index.html` - sudah disiapkan untuk menggunakan gambar
   - Ganti path gambar di HTML dengan file Anda

## üé® Alternatif: Desain Manual

Jika AI generator tidak memberikan hasil yang diinginkan, Anda bisa:

1. **Gunakan Canva/Figma:**
   - Template business card
   - Desain sesuai spesifikasi
   - Export sebagai PNG/WebP

2. **Hire Designer:**
   - Fiverr, Upwork, atau 99designs
   - Berikan spesifikasi ini sebagai brief

3. **Stock Photo:**
   - Unsplash, Pexels (gratis)
   - Shutterstock, Getty Images (berbayar)
   - Edit dengan Photoshop/GIMP untuk menambahkan teks
